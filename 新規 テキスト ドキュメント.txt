Table Panelが表示されているときは、メニューのTable Panelの左端にチェックマークを表示

# Table Panel機能
画面下部に表示するリサイズ可能なパネル：
- Node Table / Edge Table のタブ切り替え
- ソート（昇順/降順）
- テーブル⇔グラフの選択連動
- カラム表示/非表示設定
- パネル高さのリサイズ
- カラム幅のリサイズ

初回表示: 全カラムが均等幅で表示され、スクロールなしで全体が見えるようになっていません
カラム幅調整: カラム幅調整ができません
横スクロール: カラム幅を広げた結果、テーブル全体が表示領域を超える場合は横スクロールバーが表示されます
幅の維持: 一度調整したカラム幅は、テーブルを更新しても維持されます

Node Tableの_originalBg、name、labelカラムは表示対象にしない(カラム選択にも表示しない)
Edge Tableのid、interactionカラムは表示対象にしない(カラム選択にも表示しない)


Table Panelでコピー可。Ctrlキーで複数選択

Node Classが対象材料でその対象材料の発行年月日が2024年11月以降　そして
Node Classが評価方法でその評価方法の発行年月日が2025年11月以降


Table PanelのColumnsボタンと閉じるボタンの間にFloatボタンを作成
Flotボタンを押下するとFlotウインドウとしてTable Panelを外だしできるようにしてください
FlotウインドウとしてTable Panelを表示してもTable Panel機能は全て引き継ぐこととする

Table Panelの各カラム名の下にテキストボックスを作成し抽出機能を作成してください

Table Panelの閉じるボタンの左にフィルター解除ボタンを作成してください
カラム名と抽出用テキストボックスの間を白で塗りつぶししてください

Edge Tableに_originalLinColorと_originalWidthは表示しないでください。Columns選択にも表示不要です。
ヘッダーと検索ボックスまでの背景色を薄い灰色にしてください


Table Panelを表示させるとき最初は画面サイズに合わせて全てのカラムが表示させるようにしてください

Saveをクリックしたときダウンロードフォルダにダウンロードされてしまいます
Saveをクリックしたときは、Openでネットワーク図を開いていたときは、そのときと同じファイルへ上書き、Save Asされているファイルを使用しているときは、そのときと同じファイルへ上書きするようにしてください

Save処理とOpen処理を修正

ネットワーク図でノードやエッジを選択した場合、Table Panelはその選択されたノードやエッジの情報だけを表示させるようにしてください

Table PanelでNode Table側でカラムid、Edge Table側でsource,targetを非表示にしても表示されてしまします
